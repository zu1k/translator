# Translator

Translator 是使用Rust编写的翻译小工具

## 特点

- 轻量
- 自动去除多余的换行和断句，优化PDF翻译体验
- 选中即翻译(划词翻译)，专注论文阅读

## 使用说明

工具仅一个exe，启动后会驻留后台，选中文本后按 `alt+q` 唤起翻译界面，`esc`关闭界面，`ctrl+shift+d`完全退出

在界面开启的情况下，可以通过选中文本触发翻译行为，无需快捷键，俗称“划词翻译”
